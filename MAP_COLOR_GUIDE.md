# 맵 제작 색상 가이드

PNG 이미지 파일의 픽셀 색상으로 게임 오브젝트를 정의합니다.

## 기본 오브젝트

| 색상 코드 | RGB | 설명 |
|----------|-----|------|
| 회색 | (160, 160, 160) | 마리오 시작 위치 |
| 빨간색 | (255, 0, 0) | 지면 블록 |
| 파란색 | (0, 0, 255) | 일반 벽돌 (슈퍼 마리오만 깰 수 있음) |
| 초록색 | (0, 255, 0) | 파이프 |
| 하늘색 | (0, 255, 255) | 굼바 |
| 분홍색 | (255, 0, 255) | 쿠파 트루파 |
| 보라색 | (160, 0, 160) | 골 깃발 |

## 배경 색상 (무시됨)

| 색상 코드 | RGB | 설명 |
|----------|-----|------|
| 검은색 | (0, 0, 0) | 빈 공간 (아무것도 생성하지 않음) |
| 갈색 | (127, 51, 0) | 배경 장식 (무시됨) |

## 물음표 블록 (아이템 타입별)

| 색상 코드 | RGB | 아이템 |
|----------|-----|--------|
| 밝은 노란색 | (255, 255, 0) | 랜덤 아이템 (레거시) |
| 주황빛 노란색 | (255, 200, 0) | 코인 (50점) |
| 짙은 주황색 | (255, 100, 0) | 슈퍼 버섯 |
| 중간 주황색 | (255, 150, 0) | 파이어 플라워 |
| 연두색 | (200, 255, 0) | 1UP 버섯 |
