# 맵 제작 색상 가이드

PNG 이미지 파일의 픽셀 색상으로 게임 오브젝트를 정의합니다.

## 기본 오브젝트

| 색상 코드 | RGB | 설명 |
|----------|-----|------|
| 회색 | (160, 160, 160) | 마리오 시작 위치 |
| 빨간색 | (255, 0, 0) | 지면 블록 |
| 파란색 | (0, 0, 255) | 일반 벽돌 (슈퍼 마리오만 깰 수 있음) |
| 초록색 | (0, 255, 0) | 파이프 |
| 하늘색 | (0, 255, 255) | 굼바 |
| 분홍색 | (255, 0, 255) | 쿠파 트루파 |
| 보라색 | (160, 0, 160) | 골 깃발 |

## 배경 색상 (무시됨)

| 색상 코드 | RGB | 설명 |
|----------|-----|------|
| 검은색 | (0, 0, 0) | 빈 공간 (아무것도 생성하지 않음) |
| 갈색 | (127, 51, 0) | 배경 장식 (무시됨) |

## 물음표 블록 (아이템 타입별)

| 색상 코드 | RGB | 아이템 |
|----------|-----|--------|
| 밝은 노란색 | (255, 255, 0) | 랜덤 아이템 (레거시) |
| 주황빛 노란색 | (255, 200, 0) | 코인 (50점) |
| 짙은 주황색 | (255, 100, 0) | 슈퍼 버섯 |
| 중간 주황색 | (255, 150, 0) | 파이어 플라워 |
| 연두색 | (200, 255, 0) | 1UP 버섯 |

## 숨겨진 아이템 블록

| 색상 코드 | RGB | 설명 |
|----------|-----|------|
| 파란색 (1픽셀 차이) | (0, 0, 254) | 일반 블록처럼 보이지만 코인 포함 (100점) |

## 사용 예시

### 포토샵/GIMP에서 맵 제작
1. 새 이미지 생성 (권장: 200x15 픽셀)
2. Pencil Tool 사용 (안티앨리어싱 OFF)
3. 위 색상표를 참고하여 픽셀 단위로 배치
4. PNG로 저장

### 주의사항
- **정확한 RGB 값** 사용 필수 (1픽셀이라도 다르면 인식 안 됨)
- 안티앨리어싱, 그라디언트 사용 금지
- 파일 포맷: PNG (JPG 사용 금지)

## 맵 배치 팁

### 균형잡힌 아이템 배치
```
- 슈퍼 버섯: 초반에 1-2개 (플레이어 성장 유도)
- 파이어 플라워: 중반 이후 배치
- 1UP 버섯: 숨겨진 블록으로 배치 (탐험 보상)
- 코인: 많이 배치 (점수 획득용)
```

### 난이도 조절
```
- 쉬움: 슈퍼 버섯 3-4개, 파이어 플라워 2-3개
- 보통: 슈퍼 버섯 2개, 파이어 플라워 1-2개
- 어려움: 슈퍼 버섯 1개, 파이어 플라워 1개
```

## 기존 맵 업데이트

기존 맵 파일 (`Map 1.png`, `Map 2.png`)은 **RGB(255, 255, 0)** 랜덤 블록을 사용합니다.
- 그대로 사용 가능 (랜덤 시스템 유지)
- 원하는 경우 새 색상 코드로 교체하여 고정 아이템 배치 가능
